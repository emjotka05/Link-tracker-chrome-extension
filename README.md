# Link-tracker-chrome-extension
Link Saver Pro
A Chrome extension for saving URLs and text notes directly to the browser's local storage.

# Features
Save the current active tab URL.

Save custom text inputs (descriptions or notes).

Persistent data storage.

# Local Installation
1. Download the project as a ZIP file and extract it to a local folder.

2. Open Google Chrome and navigate to chrome://extensions.

3. Toggle Developer mode on in the top right corner.

4. Click Load unpacked in the top left corner.

5. Select the extracted project folder (ensure it contains the manifest.json file).

# Usage
SAVE INPUT: Type text into the input field and click this button to save it as a note.

SAVE TAB URL: Click to automatically fetch and save the current tab's link.

DELETE ALL: Click this button to permanently clear all saved data.
